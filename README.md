# .NET Курс - Блок 1: Основы языка C#

## Описание
Этот репозиторий содержит практические задания для блока 1 курса по C#. Цель блока - освоить базовые конструкции языка, принципы ООП и работу с коллекциями.

## 🎯 Цели обучения
- Освоить синтаксис C#: типы, переменные, операторы, управление потоком
- Научиться работать с коллекциями и LINQ
- Понимать принципы ООП: классы, интерфейсы, наследование, композиция
- Освоить обработку ошибок и исключений
- Научиться работать со строками и датами

- [Задание 1.2: Коллекции и LINQ](./src/Basics/Tasks/CollectionTasks.cs)
- [Задание 1.3: ООП в C#](./assignment-1.3.md)

В заданиях 1, 2, 4, 5 нужно написать такую логику, чтобы проходили тесты.
В задании 3 нужно написать тесты на написанную логику самостоятельно.

Для запуска тестов используем `dotnet test`

## 🏃‍♂️ Как начать работу
1. Убедитесь, что выполнили задания из Блока 0
2. Создайте новую ветку для работы: `git checkout -b feature/task-1`
3. Выполните задания
4. В конце создайте Pull Request для проверки
